### Root
    localhost/main/assets 

### Install needed

     XAMPP
     Node.js
     shop_system.sql
     
> [!TIPS]
> Start Apache and MySQL via the XAMPP Control Panel before running the project.

> [!NOTE]
> You must import shop_system.sql into your MySQL database using phpMyAdmin.


### Cmd

Install Package 

    npm install 
    
Active node server

    cd assets
    node node.js
